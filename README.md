# Brazilian Tour API

Node API with typescript for Brazilian Tour Project.

## initializing

### Requirements:

- Vscode
- Node
- Express
- Git

### Run the project:

    git clone https://github.com/code043/node-ts-api-brazilian-tour.git

###

    cd node-ts-api-brazilian-tour

###

    npm install

###

    npm run start
